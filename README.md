# Portfólio Pessoal - Atividade Prática (Fundamentos da Programação Web)

Projeto desenvolvido com `HTML5`, `CSS3` e `JavaScript` puro, sem frameworks.

## Estrutura do projeto

- `index.html`: página principal com navegação por âncoras e 4 seções obrigatórias.
- `styles.css`: identidade visual premium, layout responsivo e temas claro/escuro.
- `script.js`: interações, menu mobile, validação de formulário e simulação de envio.

## Requisitos da atividade atendidos

- [x] Tema: portfólio pessoal (currículo online)
- [x] Mínimo de 4 seções com menu visível e funcional
- [x] Seção **Sobre mim**
- [x] Seção **Formação**
- [x] Seção **Portfólio** com links de projetos
- [x] Seção **Contato** com formulário (`nome`, `e-mail`, `mensagem`)
- [x] Validação obrigatória em JavaScript
- [x] Simulação de envio com limpeza dos campos e mensagem de sucesso
- [x] Responsivo para desktop e mobile
- [x] Código organizado e comentado em pontos relevantes

## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub com o nome desejado.
2. Envie os arquivos deste projeto para a branch `main`.
3. No GitHub, abra `Settings` > `Pages`.
4. Em `Build and deployment`, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main` e pasta `/ (root)`
5. Salve e aguarde o link público ser gerado.

## Antes de entregar no AVA

1. Atualize os dados pessoais no `index.html` (nome, formação e textos).
2. Troque os links dos projetos na seção Portfólio pelos seus links reais.
3. Teste todos os links e o formulário.
4. Gere os prints das seções no navegador com a URL pública visível.
5. Monte o PDF final com:
   - Link do site publicado
   - Link do repositório público
   - Prints das páginas/seções
